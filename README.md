## Usage
#### Environment Setup
**1. Create a Conda environment:**
```shell
conda create -n MEM-VIT python==3.10.0
pip install -r requirements.txt
```
**2. Clone and Install this repository:**
```bash
git clone 
cd MEM-VIT
```
### Prediction
```shell
python predict.py 
```
### Training
**. Run Training Script:**
```shell
python train.py
```
